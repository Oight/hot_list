### 热搜

第一版只支持显示B站热搜,后面可能会加个抖音,知乎之类的(

#### 后端

- Python
- Flask

#### 前端

AI写的()

Vue + Axios

做了一点毛玻璃(

#### 本地部署

```
uv pip install flask requests flask_cors
uv run python spider.py
```

然后直接用浏览器打开项目根目录下的 `index.html` 文件，即可看到实时热搜页面
